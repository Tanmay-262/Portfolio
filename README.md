# Tanmay Jain | Software Engineer & AI/ML Enthusiast

A high-end, production-grade professional developer portfolio designed to showcase engineering expertise, AI/ML projects, and full-stack development skills. Built with a focus on performance, accessibility, and modern aesthetics.

🚀 **Live Demo:** [https://tanmayjain-portfolio.vercel.app](https://tanmayjain-portfolio.vercel.app)

---

## ✨ Features

- **Modern UI/UX:** Sleek, high-end design with curated color palettes and smooth gradients.
- **Dynamic Interactions:** Subtle micro-animations using Framer Motion for an engaging experience.
- **Dark Mode Optimized:** Seamless theme switching and premium dark mode aesthetics.
- **Responsive Design:** Fully optimized for all screen sizes (mobile, tablet, and desktop).
- **Data-Driven Architecture:** Modular components and a centralized data management system.
- **Performance Focused:** Built with Next.js for lightning-fast load times and SEO optimization.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** [Vercel](https://vercel.com/)
- **Version Control:** Git & GitHub

## 🚀 Getting Started

### Prerequisites

- [Node.js 18.x](https://nodejs.org/) or higher
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Tanmay-262/Portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📂 Project Structure

```text
├── app/               # Next.js App Router (pages and layouts)
├── components/        # Reusable UI components and sections
│   ├── layout/        # Navigation and Layout components
│   ├── sections/      # Individual portfolio sections (Hero, About, etc.)
│   └── ui/            # Atomic UI components
├── data/              # Centralized data for projects, skills, etc.
├── public/            # Static assets (images, PDFs)
└── lib/               # Utility functions and helpers
```

## 🔗 Deployment

This project is configured for automated deployments via **Vercel**. Any push to the `main` branch will trigger a production build.

To deploy manually via CLI:
```bash
npx vercel --prod
```
